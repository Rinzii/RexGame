# RexGame

Game/content repository for RexEngine.

- Engine/runtime code lives in the `RexEngine` git submodule.
- Game-specific code lives in `Content.Shared`, `Content.Client`, and `Content.Server`.
- Clone with submodules: `git clone --recurse-submodules <repo-url>`
- If already cloned: `git submodule update --init --recursive`
